# Security Policy

## Reporting a Vulnerability

This repository ships Home Manager module definitions — it contains no secrets
and runs no services. If you nonetheless find a security-relevant issue (for
example, a module that would write world-readable credentials to disk), please
report it privately:

- Use GitHub's [private vulnerability reporting](https://github.com/azusachino/harus-config/security/advisories/new), or
- Open a minimal issue that describes the impact without publishing an exploit.

Please do not open a public issue with exploit details before a fix is
available. You can expect an initial response within a few days.
